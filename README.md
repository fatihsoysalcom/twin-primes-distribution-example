# Twin Primes Distribution Example

This example Python script identifies twin prime numbers up to a specified limit and demonstrates their distribution. It calculates and displays the total count of twin primes found, and also shows cumulative counts at different milestones, providing a basic insight into how their frequency changes across number ranges. This data could be used by an AI to discover underlying mathematical patterns, such as an exponential law.

## Language

`python`

## How to Run

Save the code as main.py.
Run from your terminal: python main.py
